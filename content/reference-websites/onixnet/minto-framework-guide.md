# Minto Pyramid Framework - Content Writing Guide
**Purpose:** Apply the Minto Pyramid Principle to all Radiant Digital website content for maximum clarity and persuasion

---

## What Is the Minto Pyramid?

The Minto Pyramid Principle (Barbara Minto, McKinsey) structures communication top-down:
- **Lead with the answer/conclusion first**
- **Support with key arguments**
- **Back each argument with evidence**

```
         ┌─────────────┐
         │  MAIN POINT  │  ← The answer/recommendation
         └──────┬───────┘
     ┌──────────┼──────────┐
     ▼          ▼          ▼
┌─────────┐┌─────────┐┌─────────┐
│ Key Arg ││ Key Arg ││ Key Arg │  ← Supporting arguments (3 max)
│    1    ││    2    ││    3    │
└────┬────┘└────┬────┘└────┬────┘
     ▼          ▼          ▼
  Evidence   Evidence   Evidence   ← Data, metrics, case studies
```

## How to Apply Minto to Website Sections

### Hero Section (The Main Point)
```
MINTO: Lead with the single most important message

BEFORE (weak):
  "Radiant Digital offers a range of digital transformation services
   including AI, cloud, and product development."

AFTER (Minto-applied):
  "We help enterprises achieve 40% faster digital transformation
   through AI-first solutions — trusted by Fortune 500 leaders."
```

The hero = your conclusion. The rest of the page = your supporting arguments.

### Service Sections (Key Arguments)
Each service section should follow SCQA:

**S** - Situation: What's the current state?
**C** - Complication: What's the problem or challenge?
**Q** - Question: What needs to be solved? (implied)
**A** - Answer: How we solve it + proof

```
EXAMPLE - Cloud Transformation Section:

SITUATION:    "Enterprises run on complex, multi-cloud infrastructures."
COMPLICATION: "Legacy systems create bottlenecks — 60% of cloud migrations
               exceed budget and timeline."
QUESTION:     (implied: How do you migrate without the pain?)
ANSWER:       "Our Cloud Transformation practice delivers 45% cost reduction
               and 99.9% uptime — proven across 200+ enterprise migrations."
```

### Metrics Sections (Evidence)
Metrics ARE the evidence layer. Structure them to support the key arguments:

```
KEY ARGUMENT: "AI-first solutions deliver measurable ROI"

EVIDENCE:
  ├── 35% cost savings through AI automation
  ├── 70% increase in operational efficiency
  ├── 30% faster time-to-insight
  └── 4.8/5 client satisfaction score
```

## Minto Rules for Web Content

### Rule 1: Answer First, Explain Second
- Headlines state the conclusion
- Body copy provides the reasoning
- Metrics provide the proof
- **Never** make the reader scroll to find your point

### Rule 2: Group in Threes
- 3 key value propositions
- 3 service pillars
- 3 proof points per claim
- The human brain processes groups of 3 most effectively

### Rule 3: MECE (Mutually Exclusive, Collectively Exhaustive)
- Each section should cover a distinct aspect
- Together, all sections should tell the complete story
- No overlap, no gaps

### Rule 4: So-What Chain
Every statement should survive the "So what?" test:
```
"We use AI in our solutions."          → So what?
"AI reduces manual processing by 70%." → So what?
"You save $2M annually and redeploy    → NOW you have my attention.
 talent to higher-value work."
```

## Minto Applied: Page-Level Structure

### Homepage
```
MAIN POINT (Hero):
  "AI-first digital transformation that delivers measurable outcomes"

KEY ARGUMENT 1 (Services):
  "Six integrated service areas covering the full transformation lifecycle"
  └── Evidence: Service metrics, capability breadth, tech stack

KEY ARGUMENT 2 (Results):
  "Proven impact with Fortune 500 and federal clients"
  └── Evidence: Case study metrics, client logos, testimonials

KEY ARGUMENT 3 (Differentiators):
  "Agentic AI capabilities that go beyond traditional consulting"
  └── Evidence: BCG-aligned offerings, outcome-based models, AI-native talent
```

### Service Page
```
MAIN POINT (Hero):
  "Cloud transformation that cuts costs 45% while improving reliability"

KEY ARGUMENT 1: "Assessment-led approach ensures right-fit solutions"
  └── Evidence: Methodology steps, tools used, audit process

KEY ARGUMENT 2: "Migration without disruption"
  └── Evidence: Case study — enterprise migrated in 12 weeks, zero downtime

KEY ARGUMENT 3: "Ongoing optimization, not just lift-and-shift"
  └── Evidence: Managed services metrics, SLA guarantees, cost trend data
```

## Minto + Metrics: The Power Combination

When combining Minto structure with metrics-driven content:

1. **Headline = Conclusion with a number**
   - "35% cost reduction through AI-powered cloud transformation"

2. **Supporting arguments = Themed metric clusters**
   - Efficiency cluster: speed, automation, productivity
   - Cost cluster: savings, ROI, TCO
   - Quality cluster: uptime, accuracy, satisfaction

3. **Evidence = Specific client outcomes**
   - "Verizon achieved 5x capacity scaling through our intelligent automation platform"

## SCQA Templates for Common Sections

### "Why Choose Us" Section
```
S: Organizations need transformation partners, not just vendors.
C: 60% of digital transformations fail to deliver expected ROI (McKinsey).
Q: How do you ensure your transformation actually delivers?
A: Radiant Digital's outcome-based model means we're accountable for results —
   with 40% average improvement in client KPIs and engagement structures
   tied to measurable business outcomes.
```

### "Our Approach" Section
```
S: Every enterprise has unique challenges and legacy systems.
C: One-size-fits-all solutions waste budget and miss critical requirements.
Q: How do you tailor the approach without losing efficiency?
A: Our accelerator frameworks — CX, Product, and Sales — combine
   proven methodologies with AI-powered customization, delivering
   40% faster time-to-market while maintaining enterprise-grade quality.
```

### "Agentic AI" Section
```
S: AI adoption is accelerating — 75% of enterprises want providers to implement priority use cases.
C: Most providers deliver only 6-15% efficiency gains; enterprises expect 30-40%.
Q: Who can bridge the gap between AI potential and real-world outcomes?
A: Radiant Digital's agentic AI practice builds, deploys, and governs AI agents
   that operate within enterprise workflows — with human oversight, audit trails,
   and outcome-linked engagement models.
```
