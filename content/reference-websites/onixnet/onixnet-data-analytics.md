## Content Writing Patterns

### Data Analytics Headline Formula
```
"[Service] by [Company] — [Value Verb] the Value of [Asset]"
```
- "Maximize the Value of Data" — positions data as an underutilized asset
- Implies the client already has the data — they just need help unlocking it
- "Modernization strategies" — signals freshness, not legacy thinking

### Key Messaging Angles
- **Data as asset:** Not a cost center but a value source
- **Modernization:** Current state is outdated; transformation is needed
- **Expert-led:** "Expert data modernization strategies" — human expertise + data
- **Business transformation:** Analytics as a vehicle for business change

### Content Structure
1. **Hero:** "Maximize the value of data" — answer-first
2. **Data challenges:** Why current approaches fall short
3. **Modernization framework:** Steps to data maturity
4. **Analytics capabilities:** BI, real-time, predictive, prescriptive
5. **Industry applications:** Sector-specific data use cases
6. **Results:** Metrics dashboard showing client outcomes
7. **CTA:** Data assessment or consultation offer

### Key Takeaway for Radiant
- Data pages should frame data as an "asset to unlock" not a "problem to solve"
- Use "modernization" language — it implies forward movement
- Pair analytics services with visualization (show what insights look like)
- Lead with "Maximize" or "Unlock" — aspirational verbs that imply untapped potential
- Connect data services to decision-making — "data-driven decisions" is the real outcome
