## Page Overview

**Meta:** "Welcome to Onix's blog, your go-to source for cutting-edge insights, industry news, and expert advice"

## Content Writing Patterns

### Blog Page Headline Formula
```
"[Company] Blog — Explore the Latest [Content Type 1], [Content Type 2], and [Content Type 3]"
```
- "Explore" — invites discovery, not passive reading
- Three content types: Insights + Trends + Updates — covers depth, currency, and news

### Meta Description Pattern
```
"Welcome to [Company]'s blog, your go-to source for [value 1], [value 2], and [value 3]"
```
- "Go-to source" — positions as authoritative
- "Cutting-edge insights" — implies freshness and depth
- "Expert advice" — establishes authority

### Content Structure
1. **Hero:** Blog title + description
2. **Featured article:** Large card with image, title, excerpt
3. **Category filters:** By topic, service area, content type
4. **Article grid:** 3-column card layout
5. **Per-article card:** Image + Category tag + Title + Excerpt + Date + Read time
6. **Newsletter CTA:** Email subscription embedded
7. **Pagination:** For older content

### Key Takeaway for Radiant
- Blog/Insights pages should position the company as a thought leader
- Use "Insights" or "Perspectives" — more authoritative than "Blog"
- Featured article at top creates editorial curation feeling
- Category filters help users self-select relevant content
- Include read time on each card — signals respect for reader's time
- Newsletter CTA should be integrated naturally, not as a popup
