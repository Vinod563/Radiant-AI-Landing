## Product Overview
CAD-LITE is a cloud-hosted computer-aided dispatch (CAD) solution developed in partnership with Snohomish County 911. Scalable, secure dispatch capabilities accessible from any location without software installation.

## Core Features
- Dispatch Services — Emergency call management
- Call for Service Logging — Incident documentation
- Geo-Validation — Location accuracy verification
- Jurisdiction & Beat Identification — Geographic assignment
- Assignment Recommendations — Automated unit suggestions
- Incident Classification — Type and priority designation
- Unit Assignment & Tracking — Resource management
- RapidSOS Integration — Emergency services compatibility

## Key Capabilities
- **Command Line Interface** — 200+ CLI commands for streamlined operations
- **Real-time Data Delivery** — Lossless, verified information flow
- **Cross-center Overflow Support** — Call sharing between 911 centers
- **Role-Based UI** — JSON-templated customization for dispatchers
- **Integrated Map View** — County location and POI overlays
- **Real-time Alerting** — Visual alerts with custom color schemes

## Use Cases
1. **Primary Operations** — Smaller call centers handling routine dispatch
2. **Backup/Secondary** — Continuity during outages, maintenance, evacuations

## Technical Architecture
- Serverless AWS stack (API Gateway, Lambda, Aurora Serverless MySQL)
- S3 static website hosting
- Solace publish/subscribe framework
- Hosted on AWS GovCloud for government compliance
