## Design System

### Colors
- **Background:** Black (dominant), White (text/sections)
- **Accent:** Purple/Pink gradient (`#dc5adb` range)
- **Decorative:** Curved crescent shapes, abstract rings in gradient

### Typography
- **Primary Font:** Figtree (Regular, 500, 600, 700)
- **Accent Font:** Playfair Display (Regular, 500, 600, 700, Italic)

### Animations
- Scroll-triggered text splits (GSAP/SplitType)
- Fade transitions
- Rotating hero component
- Subtle glow pulses
- Blur filters, 3D transforms

### Components
- Pill/badge tags for categories
- Card grid with hover image states
- Testimonial carousel (auto-advance)
- Sticky header
- Hamburger menu on mobile
