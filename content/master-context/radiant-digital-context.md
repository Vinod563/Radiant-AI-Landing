## 18. UI Component Patterns

### Buttons

- **Primary CTA:** `#91C46B` background, white text, `border-radius: 30px`, `padding: 12px 24px`
- **Secondary:** Transparent background, `#588596` border, dark text
- **Icon Button:** Circular, `border: 2px #DBEEFF`, `border-radius: 50%`
- **Hover:** Scale transform `1.05–1.2`, elevated shadow, darker green tone

### Cards

- White background, `border-radius: 16px`, subtle box-shadow
- Hover: Elevated shadow, optional scale `1.02`
- Padding: `24px–32px`

### Navigation

- Fixed horizontal header, white background, 1px bottom border
- Logo-centered with flanking menu items
- Mega-menu dropdowns with shadow
- Mobile: Hamburger toggle

### Sections

- Alternating white / `#F4F9FC` / `#09465D` backgrounds
- Generous vertical padding (`60px–100px`)
- Constrained content width with centered alignment
